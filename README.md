# Test-project-for-Sarafan-Doct24 

## Описание проекта
Основные функции:
- показывать список карточек
- добавлять новую карточку
- показывать по клику дополнительную информацию по каждой карточке

## Как запустить проект
1. `git clone https://github.com/MaryKhoroshikh/Test-project-for-Sarafan-Doct24.git`.
2. npm install
4. npm start

## Технологии и инструменты
- Языки программирования: JavaScript.
- Фреймворки и библиотеки: React, Redux, react-icon, Formic, Yup.
